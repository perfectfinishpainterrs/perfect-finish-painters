DROP YOUR IMAGES HERE
=====================

Drop images into this folder to replace the website photos.

How it works:
  - Name your files to match what you want to replace:
      project-1.jpg  ->  replaces portfolio image #1 (Shed Transformation)
      project-2.jpg  ->  replaces portfolio image #2 (Room Renovation)
      project-3.jpg  ->  replaces portfolio image #3 (Commercial Ceiling)
      project-4.jpg  ->  replaces portfolio image #4 (Radiator Refinishing)
      project-5.jpg  ->  replaces portfolio image #5 (Basement Floor)
      project-6.jpg  ->  replaces portfolio image #6 (Whole House Interior)
      project-7.jpg  ->  replaces portfolio image #7 (Ceiling Restoration)
      logo.png       ->  replaces the site logo
      logo.jpg       ->  replaces the alt logo

  - OR just drop any images in and run the swap script:
      python swap-images.py

    The script will show you the current images, let you pick which
    slot to replace, and handle the renaming + copying automatically.

Supported formats: .jpg .jpeg .png .webp
